- **Authors:** Blaise Agüera y Arcas, Jyrki Alakuijala, James Evans, Ben Laurie, Alexander Mordvintsev, Eyvind Niklasson, Ettore Randazzo, Luca Versari

- **arXiv:** https://arxiv.org/abs/2406.19108
